# 📊 TalaLiveBot - ربات قیمت لحظه‌ای طلا و ارز

ربات گزارش لحظه‌ای قیمت طلا، سکه، ارز دیجیتال و ... برای **تلگرام، بله و روبیکا**.

## ✨ ویژگی‌ها
- قیمت لحظه‌ای از سایت tala.ir
- پشتیبانی از تلگرام + بله + روبیکا
- تاریخ شمسی
- ایموجی هوشمند
- کاملاً ماژولار و قابل گسترش

## نصب و اجرا

```bash
git clone https://github.com/arashqz/TalaLiveBot.git
cd TalaLiveBot
pip install -r requirements.txt
cp .env.example .env
# توکن‌ها را پر کنید
python bot.py
```
